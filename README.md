# Microservice User
CRUD API for User Entity
1. GET /users: Retrieve a list of all users.
2. GET /users/{userId}: Retrieve details of a specific user. 
3. POST /users: Create a new user. 
4. PUT /users/{userId}: Update details of an existing user. 
5. DELETE /users/{userId}: Delete a specific user.


# How to run project 
mvn spring-boot:run
